## 🎨 QUICK CUSTOMIZATIONS

### Change Your Name
Edit `index.html`, find this and change it:
```html
<h1 class="hero-title">Hi, I'm <span class="accent">YOUR NAME</span></h1>
```

### Change Colors
Edit `style.css`, find this and change hex codes:
```css
:root {
    --primary-color: #00d4ff;      /* Change this to any color */
    --secondary-color: #00ffaa;    /* Like #FF5733 or #9C27B0 */
    --accent-color: #ff006e;
}
```

### Add New Project
Copy this block in `index.html` and edit:
```html
<div class="project-card">
    <div class="project-icon">
        <i class="fas fa-star"></i>
    </div>
    <h3>Your Project Name</h3>
    <p class="project-type">Category | Type</p>
    <p class="project-description">Your description here</p>
    <div class="project-tech">
        <span>Tech1</span>
        <span>Tech2</span>
    </div>
</div>
```

### Update Contact Info
Find and edit these in `index.html`:
```html
<a href="mailto:your-email@gmail.com">
<a href="tel:+91YOUR_NUMBER">
<a href="https://linkedin.com/in/YOUR_PROFILE">
```
